Spotify Uniquefier

It compares two Spotify playlists and creates a new playlist containing only songs which don't appear
in both, allowing you to discover new music in another playlist without having to skip through things you
already listen to regularly. The playlist can update without having to create a new playlist each time.



GitHub Data Visualiser

I produced a program using Python where it gets the Top 100 Repositories on GitHub in order of most
starred and plots them onto a plot graph that has an interactive control. It displays whether or not there
is any correlation between the numbers of stars versus the number of commits the repository has had in the
present year. The graph is then displayed onto a basic HTML website that contains hyperlinks for the Top 10
Starred Repositories on GitHub.
